# Django study

## 강의 

- Python Web Framework 중 하나인 Django의 기본 개념과 간단한 실습을 진행한다.
- 생활코딩의 [강의](https://opentutorials.org/course/4886) 를 따라가며 진행했다.

## 개발 환경

- Python 3.12.0
- Django 4.2.7

> Python 3.11 버전은 _sqlite3 을 찾지 못하는 버그가 계속 발생하여 3.12 버전을 사용함
